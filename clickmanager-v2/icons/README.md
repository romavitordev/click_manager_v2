# Icon Assets

Use esta pasta para armazenar as versões de logo e ícones do projeto:

- `favicon.ico`
- `favicon-32x32.png`
- `favicon-16x16.png`
- `apple-touch-icon.png`
- `android-chrome-192x192.png`
- `android-chrome-512x512.png`
- logos para redes sociais, OpenGraph e branding

> Observação: para que o navegador carregue os favicons usando os caminhos `/<nome-do-arquivo>`, copie os arquivos finais também para a raiz do site `clickmanager-v2/`.
